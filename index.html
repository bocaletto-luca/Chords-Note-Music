Chord Note Music Database

A robust, fully-populated MySQL chord library for the entire chromatic scale, bilingual (Italian & English).

Project Overview

The Chord Note Music Database is a meticulously designed MySQL schema that catalogs musical chord data for each note in the chromatic scale—covering natural notes, sharps, and flats. Each note has its own table, storing chord names and interval compositions in both Italian and English.

Authored and maintained by Luca Bocaletto, this project aims to:

  • Provide a bilingual reference for chord names and interval breakdowns.
  • Include every common chord extension up to the 13th (triads, 6ths, 7ths, 9ths, 11ths, 13ths).
  • Offer a production-ready schema with timestamps and uniqueness constraints to avoid duplicates.
  • Serve as a foundation for music education tools, chord lookup APIs, and integration into DAWs or web applications.

Key Features

Uniform Table Structure

Every table (`NoteDO`, `NoteREdiesis`, `NoteSIb`, etc.) follows the same DDL:

CREATE TABLE NoteXY (
  id               INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,
  note_ita         VARCHAR(3)    NOT NULL COMMENT 'e.g. Do♯',
  chord_ita        VARCHAR(50)   NOT NULL,
  chord_note_ita   VARCHAR(100)  NOT NULL,
  note_eng         VARCHAR(2)    NOT NULL COMMENT 'e.g. C#',
  chord_eng        VARCHAR(20)   NOT NULL,
  chord_note_eng   VARCHAR(100)  NOT NULL,
  created_at       TIMESTAMP     NOT NULL DEFAULT CURRENT_TIMESTAMP,
  updated_at       TIMESTAMP     NOT NULL DEFAULT CURRENT_TIMESTAMP 
                    ON UPDATE CURRENT_TIMESTAMP,
  UNIQUE KEY uk_ita(note_ita,chord_ita),
  UNIQUE KEY uk_eng(note_eng,chord_eng)
);
Comprehensive Data
  • All 12 chromatic notes, plus enharmonic flats/sharps.
  • Both Italian and English chord names and interval lists.
  • Full set of INSERT scripts for each chord type up to 13ths.
  • Automatic created_at / updated_at fields.

Installation & Usage

  1. Clone the repository: 
    git clone https://github.com/bocaletto-luca/Chords-Note-Music.git
  2. Create and select your database: 
    CREATE DATABASE ChordNoteMusic;
USE ChordNoteMusic;
  3. Import the full schema and data: 
    mysql -u <username> -p ChordNoteMusic < file.sql
  4. Query examples: 
    -- List all D9 chords in Italian and English
SELECT * FROM NoteRE WHERE chord_eng LIKE 'D%9%';
